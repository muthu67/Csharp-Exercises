# DotNet DSA Patterns

A comprehensive collection of Data Structures and Algorithms (DSA) solutions in C# (.NET), organized by problem types and patterns.

## Project Structure

```
DotNetDSAPatterns/
│
├── Arrays/              - Array-based problems
│   ├── TwoSum/          - Find two numbers that sum to target
│   ├── ContainsDuplicate/ - Detect duplicate elements
│   ├── ProductExceptSelf/ - Product of array except self
│
├── Hashing/             - Hash map and hash set problems
│
├── Patterns/            - Pattern documentation
│   ├── Arrays.md        - Common array patterns
│   ├── Hashing.md       - Common hashing patterns
│
├── Templates/           - Code templates
│   ├── ArrayTemplate.cs - Template for array solutions
│   ├── HashMapTemplate.cs - Template for hashing solutions
│
├── README.md            - This file
└── DotNetDSAPatterns.sln - Solution file
```

## Getting Started

1. Open `DotNetDSAPatterns.sln` in Visual Studio or VS Code
2. Navigate to the problem folder you want to solve
3. Use the templates in the `Templates/` folder as reference
4. Refer to pattern documentation in `Patterns/` folder

## Patterns Covered

### Arrays
- Two Sum
- Contains Duplicate
- Product Except Self
- Sliding Window
- Two Pointers
- Prefix Sum

### Hashing
- Frequency Counter
- Hash Map Lookup
- Group Anagrams
- Valid Anagram

## How to Use Templates

1. **ArrayTemplate.cs** - Use this as a starter for array problems
2. **HashMapTemplate.cs** - Use this as a starter for hashing problems

Each template includes:
- Problem setup steps
- Edge case handling
- Common helper methods
- Implementation guidance

## Contributing

Add new problem solutions by:

3. Adding unit tests
4. Documenting the approach and complexity
